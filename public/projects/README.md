# Project media folders

Drop images or videos here. **One folder per project** (folder name = project id). Files are shown in **alphabetical order by filename**.

Supported: images (jpg, jpeg, png, webp, gif), videos (mp4, webm, mov, m4v).

**Logo (optional):** A file named **`logo.svg`**, **`logo.png`**, or **`logo.webp`** in a project folder is used as that project’s intro card logo. SVG is recommended so it scales.

**Logo size:** For short horizontal logos (e.g. Fellow, General Collaboration), add the project id to `WIDE_LOGO_PROJECT_IDS` in `App.tsx` to use the larger size (75% larger).

**Stroke (optional):** Filename before the extension ending with `-w` → white stroke; `-b` → black stroke. No suffix → no stroke.
